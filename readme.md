#avl tree
Sort program that adds and removes from avl tree
##compile
make
##usage
a [number] or i [number]  
adds a number to the tree  
k [number]  
prints kth smallest  
r [number]  
removes from tree  
c [number]  
checks if tree contaions number  
d   
prints tree  
b [number1] [number2]  
checks if between number1 and number2  
q  
quits the program
